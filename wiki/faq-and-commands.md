# FAQ and Commands

## 1) What this system is

This page answers common player questions and lists the main player-facing commands and UI shortcuts.

## 2) Who should use it

- New players.
- Returning players.
- Anyone who wants a quick control and command refresher.

## 3) Core rules (exact mechanics)

This page intentionally excludes admin/mod-only commands.

Command availability depends on server permissions and your rank.

The biggest recent UI shortcut:

- `O` opens the beta inventory UI.

## 4) Recommended menu order

If you are new, this is the cleanest order:

1. `O` for beta inventory
2. `/quests`
3. `/class`
4. `/stats`
5. `/skills`
6. `/p` and `/d`

## 5) Player commands and shortcuts

- `O` - open the beta inventory UI
- `/i` - open the beta inventory UI if command access is enabled
- `/quests` (aliases often include `/q`, `/quest`) - open quest log
- `/class` (alias `/c`) - class panel/management
- `/skills` - profession panel
- `/stats` - current stat display
- `/shop` - open shop
- `/bal` - view gold balance
- `/bank` - open bank storage
- `/auction` (alias `/ah`) - auction house
- `/d` - dungeon finder
- `/dungeon leave` - leave current dungeon
- `/p` or `/party` - open party browser
- `/p create`, `/p invite <player>`, `/p accept`, `/p leave`, `/p info`
- `/leaderboard` (alias `/lb`) - leaderboard UI
- `/arena` and `/arena leave` - arena flow
- `/pet` - pet commands

## 6) Common questions

### What is the new inventory UI?

The beta inventory is a custom MMO-style inventory with:

- armor slots
- ring slots
- amulet slot
- live stat panel
- class/talent/ability view
- shared top tabs for other major menus

The shared top-tab order is:

- Inventory
- Skills
- Quests
- Party
- Dungeon Finder

### Why can I not equip this weapon?

Class weapon rules still apply:

- Fury -> battleaxes
- Light -> clubs
- Sol -> staffs
- Shinobi -> swords

You may also be under the required item level.

### Why can I not equip this armor?

Armor still respects class armor limits and required level.

### Why did my quest progress look wrong after changing character?

Quest progress is character-based. If something still looks wrong after a relog or character swap, report it.

### Why can I not queue dungeon right now?

You may be:

- below the level requirement
- in a ready-check penalty window
- in an invalid party state
- trying to queue with a member who does not meet requirements

## 7) Common mistakes

- Forgetting that `O` now opens the main beta inventory.
- Not checking `/skills` after ignoring professions for too long.
- Ignoring jewelry slots and food/potion prep.
- Thinking all menu flow still has to go through commands only.
