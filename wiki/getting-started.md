# Getting Started

Back: [Home](../index.md)

## 1) What this system is

This page is your first-session route so you do not waste time or pick a path blind.

Rift Haven starts with story quests, then opens class progression, professions, consumables, jewelry, utility systems, and early group content.

## 2) Who should use it

- Brand-new players
- Returning players on fresh characters
- Anyone helping a friend through the early game

## 3) Core rules (exact mechanics)

- You start with story-driven onboarding, not a random grind loop.
- Your class path controls weapon restrictions and early power.
- Class talents unlock at levels 5, 10, 15, 20, 25, 30.
- Profession level cap is 100.
- Important early controls:
- `F` interact
- Left-click basic attack
- Hotbar keys use equipped abilities/items
- `O` opens the beta inventory UI

Useful early UIs:

- `O` for beta inventory
- `/i` if command access is enabled
- `/quests` for quest log
- `/class` for class info
- `/skills` for profession progression
- `/stats` for stat snapshot

## 4) Recommended first route

1. Complete `Knocking on Haven's Door`
2. Complete `Meet the Trainers`
3. Complete `Choose Your Devotion`
4. Pick your class weapon quest:
- Fury Fighter -> `And My Axe`
- Light Paladin -> `The First Light`
- Sol Elementalist -> `A Bright Tomorrow`
- Shinobi Rogue -> `Ninja Tools`
5. Open the beta inventory with `O` and get used to the menu flow
6. Start nearby town chains:
- miner/blacksmith path
- farmer/weaver path
- quartermaster and fishmaster/artificer side chains
7. Start gathering early:
- Mining for bars/gear
- Foraging for cooking/alchemy materials
8. Keep useful creature drops:
- essences
- hides
- meat
- gems
9. Start food, potion, and jewelry progression as soon as you can support it

## 5) Early priorities

- Learn your class weapon identity fast
- Start professions early instead of delaying them
- Keep an eye on ring/amulet slots once jewelry starts becoming available
- Use food and potions once you can make or buy them
- Do not throw away essences if you plan to use Alchemy, Jewelcrafting, or permanent essence progression later

## 6) Common mistakes

- Ignoring the `O` inventory and missing how much information it now gives you
- Selling all essences/materials immediately
- Waiting too long to start Foraging, Cooking, or Alchemy
- Ignoring chest loot because dungeon/material progression is more useful now than it used to be
- Joining group content with no consumable prep
